Privacy Policy — Giovanni Lazuro

Last updated: 2026-05-19

1. Scope

This privacy policy applies to the application operator-pin, operated by Johannes Liebl under the brand name Giovanni Lazuro, used to automate publishing of original content to the Pinterest account giovannilazuro.

The application is a personal automation tool. It is not offered to third-party users.

2. Data the application collects

The application does not collect, store, or process personal data from third-party users. The only data the application reads is content from the Pinterest account owned by Johannes Liebl, accessed via Pinterest's official OAuth flow.

3. Storage

OAuth tokens issued by Pinterest are stored locally on the operator's personal Windows machine and are not transmitted to any third party.

4. Data sharing

No data of any kind is shared with third parties. The application makes API calls only to Pinterest's own servers (api.pinterest.com).

5. Cookies, analytics, tracking

The application does not use cookies. It does not run analytics, telemetry, or tracking of any kind. It does not run advertising. It does not perform remarketing.

6. Children's privacy

The application is not directed at children under 13. No data from minors is collected.

7. Your rights (GDPR)

As the application does not process third-party personal data, no GDPR data-subject requests are applicable. The owner's own data is handled by Pinterest directly under Pinterest's privacy policy.

8. Contact

Privacy questions: johannes.liebl@gmx.de

9. Changes to this policy

If material changes are made to this policy, the updated date at the top will be revised. Continued use of the application after changes implies acceptance.
